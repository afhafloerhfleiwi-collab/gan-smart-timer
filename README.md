# 🧠 GAN Smart Timer (csTimer-like)

A web-based speedcubing timer with:
- 3D Rubik’s Cube
- GAN Smart Cube Bluetooth support
- WCA scramble generator
- Session statistics (Ao5 / Ao12)
- Inspection + timer system

---

# 🚀 Features

## ⏱ Timer
- Spacebar start / stop
- 15-second inspection
- csTimer-like behavior

## 🎲 Scramble
- WCA-compliant 3x3 scrambles
- No invalid patterns:
  - No `U U'`
  - No `R R`
  - No same-face repetition

## 🧊 3D Cube
- Built with Three.js
- Fully interactive rotation
- Real cube structure (27 cubies)
- WCA color scheme

## 🔵 GAN Smart Cube (Experimental)
- Web Bluetooth connection
- Supports GAN BLE devices (GAN356 / GAN12 / GAN14)
- Move parsing framework
- Real-time event system (basic)

⚠️ Note: GAN protocol is partially proprietary. This implementation is a simplified model.
